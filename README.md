## Description

Web Application to handle bazar sales and inventory

## Tecnologies

- TypeScript
- React
- Next.js
- ShadCN

## Requirements

- Node.js ^22.13.1
- bun

## Installation

1. Install dependencies:

```bash
$ bun install
```

2. Run application:

```bash
# development watch mode
$ bun dev

# production mode
$ bun build
$ bun start
```
